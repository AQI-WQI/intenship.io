Hardware Used

DHT11 – Temperature & Humidity Sensor

Used to monitor the ambient temperature and humidity levels for environmental awareness and health.

MQ2 – Gas Sensor

Detects the presence of gases like LPG, methane, and smoke to ensure air quality and safety.

Turbidity Sensor

Measures the cloudiness of water, helping detect water quality and contamination levels.

ESP8266 Wi-Fi Module

Provides wireless connectivity for transmitting sensor data to online platforms like ThingSpeak.

Arduino Uno

Acts as the main microcontroller to collect sensor data and coordinate communication and control.

Software Used

Arduino IDE

An open-source development environment used to write, compile, and upload code to microcontroller boards like Arduino Uno and ESP8266.

ThingSpeak

An IoT analytics platform that collects, stores, visualizes, and analyzes real-time sensor data in the cloud.

Visual Studio Code (VS Code)

A lightweight and powerful source code editor used for designing the frontend of the website and integrating sensor data.

GitHub

A platform for version control and collaboration that helps manage code repositories and publish websites using GitHub Pages.
