About the Project

Technologies and Components Used

Arduino UNO

Microcontroller used for processing sensor inputs and sending data via ESP8266.

ESP8266 WiFi Module

Used to transmit sensor data to the ThingSpeak cloud platform.

DHT11 Sensor

Measures temperature and humidity of the environment.

MQ135 Gas Sensor

Detects harmful gases like CO2, NH3, Benzene, etc., in the air.

Turbidity Sensor

Measures the cloudiness of water, indicating water quality.

Arduino IDE

Used for writing and uploading code to the Arduino board.

ThingSpeak

Cloud platform for visualizing and analyzing live sensor data.
