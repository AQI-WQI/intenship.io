About the Project

Hardware and Software Used in Our Monitoring System

Hardware Used

DHT11

Measures temperature and humidity.

MQ135

Detects harmful gases like CO2, NH3, and Benzene.

Turbidity Sensor

Monitors water clarity or quality.

ESP8266

Wi-Fi module that transmits sensor data to the cloud.

Arduino Uno

Microcontroller board for controlling and reading sensors.

Software Used

Arduino IDE

Used to write and upload code to Arduino and ESP8266.

ThingSpeak

Cloud platform for visualizing sensor data in real-time.

Visual Studio Code (VS Code)

Used to build and edit the frontend of the website.

GitHub

Used for version control and hosting the live website.
